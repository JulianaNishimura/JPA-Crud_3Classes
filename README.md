# JPA CRUD - 3 Classes

Projeto CRUD desenvolvido em Java utilizando JPA e Hibernate para gerenciamento de entidades e persistência de dados em banco relacional.

## Tecnologias
- Java
- JPA
- Hibernate
- MySQL

## Entidades
O sistema possui relacionamento entre três entidades:
- Cliente
- Pedido
- Produto

## Funcionalidades
- Cadastro de clientes, pedidos e produtos
- Consulta de registros
- Atualização de dados
- Remoção de registros
- Relacionamento entre entidades com JPA

## Objetivo
Projeto desenvolvido para praticar modelagem de entidades, relacionamentos e operações CRUD utilizando JPA/Hibernate.
